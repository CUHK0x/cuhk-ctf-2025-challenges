Flag format: `cuhk25ctf{<latitude>_<longitude>}`.
The values should be taken from Google Maps, and rounded off to nearest 4 decimal places.
For example, if the location is at [the gilded flower bauhinia sculpture at Golden Bauhinia Square, Hong Kong](https://www.google.com/maps/place/22%C2%B017'03.5%22N+114%C2%B010'25.5%22E/@22.2842979,114.1711781,17z/data=!3m1!4b1!4m4!3m3!8m2!3d22.284293!4d114.173753?entry=ttu&g_ep=EgoyMDI1MDkxNy4wIKXMDSoASAFQAw%3D%3D).
